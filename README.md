# Projeto - Banco de Dados de Tintas Artísticas

O Projeto Banco de Dados de Tintas Artísticas foi criado no Laboratório de Espectroscopia e Espectrometria das Radiações - LEER nas instalações do Instituto de Pesquisas Energéticas e Nucleares - IPEN. E busca criar uma base de dados de tintas artísticas e pigmentos confiável e organizada contendo os dados de composição química, por meio da utilização da técnica de Fluorescência de Raios - X (FRX); assim como, informações sobre pigmentos.

A base de dados conta com buscas por nome de tintas; cores; décadas; tipos de técnicas e medium utilizados. E os resultados contêm informações gerais da respectiva tinta, como nome, década e medium, assim como, dados analíticos qualitativos que apresentam os elementos majoritários e traços encontrados, junto de gráficos com as respectivas distribuições elementais.
